INTRODUCTION
------------

The Modal Page allow create Modal in CMS and set page to show. If the 
user visit this page that configures it shows the modal.


REQUIREMENTS
------------

No special requirements


INSTALLATION
------------

Install as you would normally install a contributed Drupal. See:
https://drupal.org/documentation/install/modules-themes/modules-7 for further
information.


CONFIGURATION
-------------

 * Configure your messages in Configuration » User interface » Modal Page
 
   - Set the page name to show.
   - Set the Title of modal.
   - Set the Text of modal.
   - Set the Button label of modal.
   - Format: Page|Title|Text|Button
