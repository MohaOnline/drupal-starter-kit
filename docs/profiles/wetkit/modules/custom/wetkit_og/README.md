WxT Organic Groups
==================
Helps integrate and manage [Organic Groups][organicgroups] for [Drupal WxT][drupalwxt].

Key Features
------------

* Provides integration with [Organic Groups][organicgroups] and [Workbench Moderation][workbench_moderation].
